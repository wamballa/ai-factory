# Standard Agent Workflow

Use this workflow for all root-level agent runs.

1. Read `AGENTS.md` and `CONTEXT.md`.
2. Inspect `TODO.md`.
3. Select the highest-priority item in `Backlog`.
4. Move the selected item to `In Progress`.
5. Perform the work.
6. Move the completed item to `Completed`.
7. Record discoveries in `memory/inbox.md`.

## Notes

- `TODO.md` is the single task board.
- Do not create additional task tracking files.
