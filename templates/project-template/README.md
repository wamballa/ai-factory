# Project Template

This directory is a scaffold for new micro-SaaS projects in AI Factory.

## Included Files

- `AGENTS.md`: project-specific agent operating rules
- `CONTEXT.md`: project background, goals, and constraints
- `RUN_AGENT.md`: standard execution workflow for project tasks
- `README.md`: project overview and usage notes

## How to Use

1. Copy this template into `projects/<new-project-name>/`.
2. Replace placeholder values in `CONTEXT.md`.
3. Add project-specific details to this `README.md`.
4. Track all work in root `TODO.md` (do not create local task files).

## Suggested README Sections for New Projects

- Problem statement
- User persona
- MVP scope
- Architecture
- Delivery milestones
