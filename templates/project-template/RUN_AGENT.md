# Project Agent Workflow

Use this workflow for project-level agent runs.

1. Read `AGENTS.md` and `CONTEXT.md` in this project.
2. Inspect root `../../TODO.md`.
3. Select the highest-priority item in `Backlog` relevant to this project.
4. Move it to `In Progress`.
5. Perform the work in this project directory.
6. Move the task to `Completed`.
7. Record discoveries in `../../memory/inbox.md`.
