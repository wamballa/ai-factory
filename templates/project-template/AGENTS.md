# Project Agent Instructions

This file defines agent behavior for this project.

## Purpose

Use this project directory for implementation work specific to this micro-SaaS.

## Rules

1. Read this file, `CONTEXT.md`, and `RUN_AGENT.md` before starting work.
2. Use root `TODO.md` as the only task board for work tracking.
3. Do not create project-specific task files.
4. Store discoveries in root `memory/` files.
5. Keep project documentation in this project directory.

## Memory Usage

Record project discoveries in:

- `../../memory/inbox.md` for temporary findings
- `../../memory/knowledge.md` for stable knowledge
- `../../memory/decisions.md` for key design decisions
