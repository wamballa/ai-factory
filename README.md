# AI Factory

A monorepo for building and orchestrating AI agents using Codex CLI and n8n.

Core features:
- AI agents for research and development
- automated task execution
- shared memory and documentation
- reusable micro-SaaS project templates

## Repository Layout

- `TODO.md`: single task board for all agents and projects
- `memory/`: shared discoveries and knowledge
- `docs/`: infrastructure and operational documentation
- `projects/`: active project implementations
- `templates/project-template/`: scaffold for new micro-SaaS projects

## Project Scaffolding

When creating a new micro-SaaS project, start from `templates/project-template/` and copy:

- `AGENTS.md`
- `CONTEXT.md`
- `RUN_AGENT.md`
- `README.md`
